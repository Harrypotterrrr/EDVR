# EDVR
This repo is an unofficial tensorflow implementation of [EDVR](https://arxiv.org/abs/1905.02716)

## Prerequisite

| **Package**    | **version**  |
|----------------|--------------|
| python         |  >=3.7       |
| tensorflow-gpu |  2.3         |
| tensorflow-datasets |  3.0.0    |
| tensorboard   |  2.3.0         |
| tqdm         |  4.49.0     |
| numpy         |  1.18.5      |

please look up `requirements.txt`
